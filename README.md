## 简介

> [vue3-admin](https://gitee.com/madt/vue3-admin) 是一个基于 Vue3 + Vite + Element Plus 等技术栈构建的PC端中后台项目。



## 环境准备

> node >= 18.20.5
> 
> pnpm >= 9.15.4



## 项目启动

```bash
# 安装 pnpm
npm install pnpm -g

# 设置镜像源
pnpm config set registry https://registry.npmmirror.com

# 安装依赖
pnpm install

# 启动服务
pnpm dev

# 本地开发访问地址
http://localhost:920
```



## 项目构建

```bash
# 测试环境打包
pnpm build:test

# 生产环境打包
pnpm build:prod
```
